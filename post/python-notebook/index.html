浅拷贝和深拷贝

浅拷贝是指重新分配一块内存，创建一个新的对象，里面的元素对原对象中子对象的引用。

>>> l3 = [[1,2,3], (3,4)]
>>> l4 = list(l3)
>>> l3[0].append(0)
>>> l3
[[1, 2, 3, 0], (3, 4)]
>>> l4
[[1, 2, 3, 0], (3, 4)]
>>> id(l3[0])
4339386704
>>> id(l4[0])
4339386704
>>> l3 is l4
False
>>> l3[0] is l4[0]
True

深拷贝是指重新分配一块内存，创建一个新的对象，并将原对象中的元素，以递归的方式，通过创建新的子对象拷贝到新对象中。因此，新对象与原对象没有任何关联。

>>> import copy
>>>
>>> l5 = copy.deepcopy(l3)
>>> l5
[[1, 2, 3, 0], (3, 4)]
>>> l3[0].append(9)
>>> l3
[[1, 2, 3, 0, 9], (3, 4)]
>>> l4
[[1, 2, 3, 0, 9], (3, 4)]
>>> l5
[[1, 2, 3, 0], (3, 4)]

实例属性查找过程

1.初级印象：先查找实例有没有这个属性，再查找类有没有这个属性，最后若没有，就报错。 2.详细属性查找过程： 一.若这个实例属性出现在类或其基类dict中，而且是数据属性描述符，则调用其get方法 二.若这个实例属性出现在实例中，则调用实例中的dict[attr]，否则 三.若这个实例属性出现类或其基类中， （一）实例属性为非数据属性描述符，则调用其get方法,否则 （二）调用类中的dict[‘attr’] 四.若这个类中有getattr方法，则调用其方法 五.抛出AttributeError
